# Java Calculator

A simple console-based calculator application developed using Core Java.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Square Root
- Power
- Memory Functions (M+, M-, MR)
- Continue/Exit option

## Technologies Used
- Java
- Eclipse IDE

## How to Run
1. Download or clone this repository.
2. Open the project in Eclipse.
3. Run Calculator.java.
